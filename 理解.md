# MMR1 项目理解路线图

## 项目一句话概述
用强化学习（RL）训练多模态视觉语言模型（如 Qwen2.5-VL），核心创新是 VAS（Variance-Aware Sampling）——智能挑选训练样本，让梯度信号更稳定。

---

## 阅读顺序

### 第一步：数据格式
**文件**：`verl/utils/dataset.py` → `RLHFDataset` 类

搞清楚输入数据长什么样：
- 每条样本 = 问题文本 + 图片（可多张）+ 标准答案
- 数据格式：HuggingFace parquet / arrow
- 关键字段：`problem`、`images`、`answer`

### 第三步：奖励函数

- 逻辑相对独立，容易单独理解

### 第四步：MMR1 核心创新 VAS
**文件**：先看 `verl/curriculum/metrics.py`，再看 `verl/curriculum/samplers.py`

**VPS 分数计算**（metrics.py）：
```
VPS = OVS（结果方差分） + TDS（轨迹多样性分）

OVS = accuracy × (1 - accuracy)
      → 半对半错的题得分最高，全对/全错的题得分最低

TDS = 多条回答之间的 BLEU 相似度
      → 回答越多样，说明模型在探索，训练价值越高
```

**混合采样策略**（samplers.py）：
```
每个 batch = 50% VPS加权样本（高价值）
           + 50% 随机样本（防止过拟合）
```

---

### 第五步：配置文件
**文件**：`examples/mmr1.yaml`

看完前四步再回来看配置，所有参数都能对上：
- `data.sampling_strategy: curriculum` — 开启 VAS
- `data.curriculum_metrics: [learnability]` — 用 OVS 作为指标
- `data.curriculum_mixture_ratio: 0.5` — 50% 加权 + 50% 随机
- `data.curriculum_rollout_n: 8` — 每题生成 8 条回答
- `algorithm.adv_estimator: grpo` — 使用 GRPO 算法

## 整体架构图
```
训练入口 verl/trainer/main.py
      │
      ▼
RayPPOTrainer.fit()          ← 主控制器
      │
      ├── MixedCurriculumSampler   ← VAS 采样（MMR1 创新）
      │       50% 高VPS样本 + 50% 随机
      │
      ├── Actor + vLLM Rollout     ← 生成回答
      │
      ├── Reward Function          ← 打分
      │
      └── GRPO 更新参数            ← 强化学习
```

## 核心论文结论
- MMR1-7B 在多模态推理基准上达到 **58.4 平均分**（7B 模型 SOTA）
- VAS 相比随机采样显著提升训练稳定性和最终效果
- 开源数据集：1.6M CoT 样本 + 15k RL 训练对
