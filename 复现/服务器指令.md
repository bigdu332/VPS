## Conda 环境

```bash
source /root/miniconda3/etc/profile.d/conda.sh
```

### 克隆 base 环境到新路径（verl-mmr1）

> RL 训练用 base 环境（verl 0.2.0.dev0 + PyTorch 2.6.0+cu124），mmr1 环境无 torch 不可用。

```bash
# 1. 激活 base 并导出快照
conda activate base
conda env export > /mnt/data/ericdu/mmr1/base_env_snapshot.yaml

# 2. 用快照创建新环境（写成一行，避免反斜杠换行报错）
conda env create --prefix /mnt/data/ericdu/conda_envs/verl-mmr1 --file /mnt/data/ericdu/mmr1/base_env_snapshot.yaml

# 3. 激活新环境
conda activate /mnt/data/ericdu/conda_envs/verl-mmr1
```
